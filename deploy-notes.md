# Deploy Notes - Simulación de Despliegue Continuo con Docker

## 1. Objetivo del despliegue

Se realiza la simulación de un despliegue continuo de una aplicación Django utilizando Docker y Docker Compose, integrando una base de datos PostgreSQL.

El objetivo es validar que el sistema puede ser replicado en un entorno externo de forma automática y reproducible.
